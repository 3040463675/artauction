# 基于区块链的艺术品竞拍系统 - 运行指南

## 一、环境要求

| 软件 | 版本 | 说明 |
|------|------|------|
| Node.js | >= 18.0.0 | JavaScript 运行环境 |
| MySQL | >= 5.7 | 数据库 |
| Ganache | 最新版 | 以太坊本地区块链 |
| MetaMask | 浏览器插件 | 以太坊钱包 |

## 二、快速启动（按顺序执行）

### 步骤 1：启动 Ganache

1. 打开 Ganache 应用
2. 点击 **Quickstart** 快速启动
3. 确认 RPC 地址为 `http://127.0.0.1:7545`，链 ID 为 `1337`

### 步骤 2：配置私钥

1. 在 Ganache 界面点击账户右侧的 🔑 图标
2. 复制私钥
3. 打开 `contracts/.env` 文件，粘贴到 `PRIVATE_KEY=` 后面

### 步骤 3：安装依赖

打开终端，依次执行：

```powershell
# 智能合约
cd D:\Art auction\contracts
npm install

# 前端
cd D:\Art auction\frontend
npm install

# 后端
cd D:\Art auction\backend
npm install
```

### 步骤 4：创建数据库

打开 MySQL，执行：

```sql
CREATE DATABASE art_auction CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

然后配置 `backend/.env` 中的数据库密码。

### 步骤 5：初始化数据库

```powershell
cd D:\Art auction\backend
npm run db:init
```

### 步骤 6：部署智能合约

```powershell
cd D:\Art auction\contracts
npm run deploy:ganache
```

部署成功后，会输出合约地址。复制地址填入：

- `frontend/.env` 中的 `VITE_NFT_CONTRACT_ADDRESS` 和 `VITE_AUCTION_CONTRACT_ADDRESS`
- `backend/.env` 中的 `NFT_CONTRACT_ADDRESS` 和 `AUCTION_CONTRACT_ADDRESS`

### 步骤 7：启动项目

**终端 1 - 启动后端：**
```powershell
cd D:\Art auction\backend
npm run dev
```

**终端 2 - 启动前端：**
```powershell
cd D:\Art auction\frontend
npm run dev
```

### 步骤 8：配置 MetaMask

1. 打开 MetaMask
2. 添加网络：
   - 网络名称：`Ganache Local`
   - RPC URL：`http://127.0.0.1:7545`
   - 链 ID：`1337`
   - 货币符号：`ETH`
3. 导入 Ganache 账户（使用私钥导入）

### 步骤 9：访问系统

打开浏览器访问：http://localhost:3000

## 四、常见问题

### Q1: npm install 报错

```powershell
# 清除缓存重试
npm cache clean --force
npm install
```

### Q2: PowerShell 禁止运行脚本

以管理员身份运行 PowerShell，执行：

```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

### Q3: 连接不上 Ganache

确保 Ganache 已启动，且 MetaMask 网络配置正确。

### Q4: 数据库连接失败

1. 确认 MySQL 服务已启动
2. 检查 `backend/.env` 中的数据库配置
3. 确认数据库 `art_auction` 已创建

### Q5: 页面中文显示为问号（乱码）

原因是源文件编码不是 UTF-8。请：

1. 用 Cursor/VSCode 打开对应 `.vue` 或 `.ts` 文件
2. 看编辑器右下角编码，若为 GBK 等，点击后选「通过编码重新打开」
3. 选择 **UTF-8**，保存
4. 或在设置中把 **Files: Encoding** 设为 `utf8`，保证新建与保存均为 UTF-8
